<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get QMD running in minutes with these simple steps
  </p>

  <Steps>
    <Step title="Install QMD">
      Install globally via npm or bun:
      
      ```bash
      npm install -g @tobilu/qmd
      # or
      bun install -g @tobilu/qmd
      ```
    </Step>

    <Step title="Create a collection">
      Index your markdown files by creating a collection:
      
      ```bash
      qmd collection add ~/notes --name notes
      qmd collection add ~/Documents/meetings --name meetings
      ```
    </Step>

    <Step title="Add context">
      Add contextual descriptions to help with search relevance:
      
      ```bash
      qmd context add qmd://notes "Personal notes and ideas"
      qmd context add qmd://meetings "Meeting transcripts and notes"
      ```
    </Step>

    <Step title="Generate embeddings">
      Create vector embeddings for semantic search:
      
      ```bash
      qmd embed
      ```
      
      <Note>Models are auto-downloaded on first use (~2GB total). They're cached in `~/.cache/qmd/models/`</Note>
    </Step>

    <Step title="Search your content">
      Use any of the three search modes:
      
      ```bash
      qmd search "project timeline"           # Fast keyword search
      qmd vsearch "how to deploy"             # Semantic search
      qmd query "quarterly planning process"  # Hybrid + reranking
      ```
    </Step>
  </Steps>
</div>
